# TicTacToe-AI
TicTacToe game created in c++. It has two game modes: 
- player v. player
- player v. computer

If player v. player is chosen:
- you'll be able to go against a player, in-person

If player v. computer is chosen:
- will be given a choice of 3 possible opponents based on general computational and AI concepts
    - Random choice
    - Monte Carlo Tree Search (MCTS)
    - Minimax algorithm with alpha-based pruning
